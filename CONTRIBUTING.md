# Contributing to n8n-gtm-workflow-pack

This repository is maintained by Forma Nôrden.

If you find an error, have a suggestion, or want to contribute an improvement:

1. Open an issue describing what you found or what you want to change
2. Fork the repo and make your changes
3. Submit a pull request with a clear description of what changed and why

All contributions must follow the existing file naming and formatting conventions.
